¡Excelente idea! Convertir el script en un extractor **multi-servicio** te va a dar una velocidad brutal en el examen. En lugar de andar editando el script a cada rato, ahora solo defines una lista separada por comas y el script se encargará de procesar todo de un solo golpe, generando un archivo .out independiente para cada servicio encontrado.
He implementado la lógica para separar la cadena por comas, limpiar espacios fantasma (por si escribes "ssh, ssl" con espacio) y vaciar la memoria entre cada servicio para que los puertos no se mezclen.
## 🛠️ El Extractor Multi-Servicio (extract_services.sh)
Copia y pega este bloque directamente en tu consola. Creará el script optimizado y lo ejecutará inmediatamente sobre tus archivos de Nmap:
```bash
cat << 'EOF' > extract_services.sh
#!/bin/bash

# =================================================================
# CONFIGURACIÓN MULTI-SERVICIO (SEPARADOS POR COMAS SIN ESPACIOS)
# =================================================================
TARGET_SERVICES="ftp,ssh,ssl,http"

# 1. Buscar dinámicamente el archivo de Nmap
files=(nmap_service_version*.nmap)

if [ ! -e "${files[0]}" ]; then
    echo "[-] Error: No se encontró ningún archivo 'nmap_service_version*.nmap' en este directorio."
    exit 1
fi

FILE="${files[0]}"
echo "[+] Archivo fuente detectado automáticamente: $FILE"
echo "================================================================="

# Convertir la lista separada por comas en una matriz (array) de Bash
IFS=',' read -ra SERVICES_ARRAY <<< "$TARGET_SERVICES"

# Iterar sobre cada servicio solicitado en la variable
for service in "${SERVICES_ARRAY[@]}"; do
    # Limpiar espacios en blanco por si se coló alguno
    service=$(echo "$service" | tr -d '[:space:]')
    [ -z "$service" ] && continue

    OUT="${service}_resume.out"
    > "$OUT" # Limpiar o crear el archivo de salida específico para este servicio
    
    echo "[*] Analizando red buscando el servicio: '$service'..."

    current_ip=""
    declare -A ip_ports

    # 2. Procesar el reporte línea por línea para el servicio actual
    while IFS= read -r line || [ -n "$line" ]; do
        # Capturar la IP del Host
        if [[ "$line" =~ Nmap\ scan\ report\ for\ ([0-9.]+) ]]; then
            current_ip="${BASH_REMATCH[1]}"
            ip_ports["$current_ip"]=""
        fi

        [ -z "$current_ip" ] && continue

        # Caso A: Puertos de la tabla estándar
        if [[ "$line" =~ ^([0-9]+)/tcp[[:space:]]+open ]]; then
            port="${BASH_REMATCH[1]}"
            lower_line=$(echo "$line" | tr '[:upper:]' '[:lower:]')
            if [[ "$lower_line" =~ "$service" ]]; then
                ip_ports["$current_ip"]="${ip_ports["$current_ip"]} $port"
            fi
        fi

        # Caso B: Capturar el puerto del Service Fingerprint actual
        if [[ "$line" =~ SF-Port([0-9]+)-TCP ]]; then
            fp_port="${BASH_REMATCH[1]}"
        fi

        # Caso C: Analizar firmas crudas (Fingerprints)
        if [[ "$line" =~ ^SF: && -n "$fp_port" ]]; then
            # Si el servicio actual buscado es SSL, rastrear su tag nativo
            if [ "$service" == "ssl" ] && [[ "$line" =~ "%T=SSL%" ]]; then
                ip_ports["$current_ip"]="${ip_ports["$current_ip"]} $fp_port"
            # Para cualquier otro servicio, buscar la palabra clave en minúsculas
            elif [ "$service" != "ssl" ] && [[ "$(echo "$line" | tr '[:upper:]' '[:lower:]')" =~ "$service" ]]; then
                ip_ports["$current_ip"]="${ip_ports["$current_ip"]} $fp_port"
            fi
        fi
    done < "$FILE"

    # 3. Formatear, ordenar numéricamente y exportar resultados
    found_any=0
    for ip in "${!ip_ports[@]}"; do
        ports=$(echo "${ip_ports[$ip]}" | tr ' ' '\n' | grep -v '^$' | sort -nu | tr '\n' ',' | sed 's/,$//')
        if [ -n "$ports" ]; then
            echo "${service}-${ip}:${ports}" >> "$OUT"
            echo "   -> Detectado: ${service}-${ip}:${ports}"
            found_any=1
        fi
    done

    # Si no se encontró nada para este servicio, borramos el archivo vacío para mantener orden
    if [ "$found_any" -eq 0 ]; then
        echo "   -> [!] No se encontraron activos para el servicio '$service'."
        rm -f "$OUT"
    else
        echo "   -> [+] Resultados consolidados en: '$OUT'"
    fi
    echo "-----------------------------------------------------------------"
    
    # Destruir el mapa de memoria para evitar residuos en la siguiente iteración
    unset ip_ports
done

echo "[+] ¡Proceso de extracción múltiple finalizado!"
EOF

# Dar permisos de ejecución y correrlo
chmod +x extract_services.sh && ./extract_services.sh

```

### Beneficios para tu examen:
 1. **Limpieza total:** Como viste con ftp, si un servicio de la lista no existe en los objetivos, el script te avisa y destruye el archivo temporal para no dejarte archivos vacíos con basura en tu carpeta de trabajo.
 2. **Reutilizable:** Si más adelante encuentras bases de datos, solo editas la línea 6 y pones TARGET_SERVICES="mysql,oracle,postgres,mssql", lo corres y tendrás tus objetivos perfectamente segmentados para tus ataques dirigidos.
